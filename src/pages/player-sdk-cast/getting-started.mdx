With IBM Video Streaming SDK for Google Cast, you can cast live and recorded content to compatible devices (such as Chromecast) from your native applications using the IBM Video Streaming Player SDK for Android.

This document describes the basic steps to integrate a customized receiver into your application.

## Before you begin

- You need to register a user at [video.ibm.com].
- Your user has to be entitled to use the IBM Video Streaming SDK for Google Cast.
  - Verify this by checking [API/SDK access] on the dashboard.
- You need to [register](https://developers.google.com/cast/docs/registration) a Google Cast Application.

If you have any questions, please [contact us].
