<PageDescription>

With IBM Video Streaming SDK for Google Cast you can cast IBM Video Streaming live and recorded content from your native applications using the IBM Video Streaming Player SDK for Android.

</PageDescription>

This document describes the basic steps to integrate a customized receiver to your application.

## Before you begin

* You need a registered user at [video.ibm.com](https://video.ibm.com/).
* Your user has to be entitled to use the IBM Video Streaming SDK for Google Cast.
* You need to [register](https://developers.google.com/cast/docs/registration) a Google Cast Application.

If you have any questions, please [contact us](https://video.ibm.com/enterprise-video/contact).
