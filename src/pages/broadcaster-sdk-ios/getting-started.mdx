## Introduction

The IBM Video Streaming SDK lets you broadcast live videos in your native applications. Using the native SDK gives you full control over the broadcast process, you can build up your own user interface, define your audio and video input sources, callbacks on status changes, and many more.

This document describes the basic steps to make an iOS and macOS native app using the IBM Video Streaming SDK.

## Before you begin

*To our clients using a regional cluster -  please [contact us] for information on SDK support.*

### Account prerequisites

Before going into details, please note that document assumes the following:

*  You have a registered user at [IBM Watson Media](http://video.ibm.com/).

*  You have a basic understanding how to use [IBM Video Streaming Channel API](https://developers.video.ibm.com/channel-api/getting-started.html).

If you have questions, please [contact us].

### Development prerequisites

The IBM Video Streaming Broadcaster SDK for iOS requires iOS 9.0 or above.
The IBM Video Streaming Broadcaster SDK for macOS requires macOS 10.11 or above.
We recommend using the most recent version of Xcode.
